# A股量化交易系统

> 完整的A股量化交易系统，支持双轨交易模式（长线股自动优化策略 + 波段股自动扫描）、模拟盘/实盘切换、多数据源、Web管理面板

## 快速开始

### 1. 安装依赖

```bash
# 核心依赖
pip install -r requirements.txt

# Web界面依赖（可选）
pip install -r requirements_web.txt
```

### 2. 运行回测

```bash
# 单标的回测
python backtest/run.py --symbol 000001 --start 20240101 --end 20241231

# 多标的回测
python backtest/run.py --symbol "000001,600519,000858" --start 20240101 --end 20241231

# 切换策略
python backtest/run.py --symbol 000001 --strategy rsi
python backtest/run.py --symbol 000001 --strategy macd
python backtest/run.py --symbol "000001,600519" --strategy multi_factor

# 调整资金和滑点
python backtest/run.py --symbol 000001 --capital 500000 --slippage 0.002
```

### 3. 回测结果

回测完成后在 `results/` 目录生成：
- `equity_curve.png` - 净值曲线 + 最大回撤图
- `trades.png` - 交易信号标记图
- `monthly_returns.png` - 月度收益热力图
- `equity.csv` - 净值数据
- `trades.csv` - 交易明细

## Web管理面板 操作手册

### 启动方式

```bash
cd /root/astock_quant
python web/app.py
```

启动后浏览器访问：`http://localhost:5000`

如需外部访问：修改 `app.py` 最后一行的 `host` 参数，或直接用 `0.0.0.0`（已默认配置）。

### 面板功能总览

Web面板共 **8个标签页**，从上到下说明：

#### 1. 总览页

显示系统关键状态指标：
- 当前交易模式（模拟盘/实盘）
- 长线股数量
- 波段最大持仓数
- 选股扫描间隔
- 每日最大交易次数
- 止损比例

底部可快速切换模拟盘/实盘模式，点击"保存"后立即生效。

#### 2. 长线股管理

**用途：** 管理长期持有的股票池。系统会为每只长线股自动回测寻找最优策略，并长期跟踪做T降成本。

**操作步骤：**
1. 输入股票代码（如 `000001`）、名称（如 `平安银行`）、分配资金
2. 点击"添加"
3. 点击"运行策略优化"，输入股票代码，系统会自动回测多种策略并选出最优解
4. 优化完成后，表格中会显示最优策略名称和评分

**说明：**
- 优化结果保存在 `data/long_term_stocks.json`
- 重复添加同一代码会提示已存在
- 点击"删除"可从池中移除

#### 3. 波段股配置

**用途：** 配置自动扫描选股的参数。系统按设定间隔扫描市场，发现符合条件的股票自动买入。

**可配置项：**
| 参数 | 说明 | 建议值 |
|------|------|--------|
| 扫描间隔 | 多久扫描一次市场（秒） | 1800~7200 |
| 最大持仓数 | 同时持有的波段股数量 | 3~10 |
| 止损比例 | 波段股亏损到此比例自动卖出 | -3% ~ -8% |
| 止盈比例 | 波段股盈利到此比例自动卖出 | 5% ~ 15% |
| 选股策略 | 用哪些策略扫描（逗号分隔） | ma_cross,macd |

配置完成后点击"保存波段配置"。

#### 4. 交易限制

**用途：** 控制每日交易频次和金额，防止过度交易或异常下单。

分为两组：
- **频率限制：** 每日最大交易次数、最小交易间隔
- **金额限制：** 单笔最大金额/股数、每日交易总额、每日买卖上限

修改后点击"保存交易限制"，配置持久化到 `config/web_config.json`。

#### 5. 交易参数

**用途：** 设置交易成本计算参数，影响回测精度和实盘成本预估。

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 佣金费率 | 0.00025 | 万2.5 |
| 最低佣金 | 5 | 不足5元按5元收 |
| 印花税率 | 0.001 | 千1，仅卖出收取 |
| 过户费率 | 0.00001 | 万0.1 |
| 滑点 | 0.001 | 千1，模拟实际成交价偏差 |
| 最小交易单位 | 100 | A股1手=100股 |

#### 6. 风控配置

**用途：** 设置仓位控制和止损止盈规则，保护资金安全。

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 单票最大仓位 | 20% | 单只股票占总资金上限 |
| 总仓位上限 | 80% | 所有持仓占总资金上限 |
| 止损比例 | -8% | 单票亏损到此比例自动止损 |
| 止盈比例 | 20% | 单票盈利到此比例自动止盈 |
| 最大回撤暂停 | -15% | 账户回撤到此比例暂停所有交易 |
| 黑名单 | 空 | 黑名单中的股票永不买入 |

#### 7. 数据源管理

**用途：** 管理数据源的启用/禁用状态，测试数据源连接。

**操作步骤：**
1. 使用右侧开关启用或禁用某个数据源
2. 如需使用 Tushare，在下方输入你的 Token
3. 点击"测试连接"选择数据源和测试股票代码，验证是否能正常获取数据

**数据源说明：**
| 数据源 | 优先级 | 是否需要Token | 覆盖范围 |
|--------|--------|---------------|----------|
| AKShare | 10 | 否 | A股全量，免费 |
| Tushare | 8 | 是 | A股全量，需注册 |
| BaoStock | 5 | 否 | A股历史数据 |
| YahooFinance | 3 | 否 | 港股/美股 |

#### 8. 日志

**用途：** 查看系统运行日志。

- 可选择显示行数（100/200/500/1000）
- 点击"刷新"手动加载
- 点击"自动刷新"每5秒自动更新，再点一次停止

## 策略说明

| 策略 | 文件 | 信号逻辑 | 适用场景 |
|------|------|----------|----------|
| 双均线 | ma_cross.py | MA5上穿MA20买入，下穿卖出 | 趋势行情 |
| RSI | rsi.py | RSI<30超卖买入，>70超买卖出 | 震荡行情 |
| MACD | macd.py | DIF上穿DEA买入，下穿卖出 | 趋势行情 |
| 多因子 | multi_factor.py | 动量+波动率+成交量综合评分 | 组合选股 |

## A股特殊处理（已实现）

| 规则 | 处理方式 |
|------|----------|
| T+1 | 买入当天禁止卖出，记录buy_date |
| 涨跌停 | 涨停(>=+9.9%)不能买，跌停(<=-9.9%)不能卖 |
| 滑点 | 买入价上偏、卖出价下偏，默认千1 |
| 手续费 | 佣金万2.5(最低5元)+印花税千1(卖出)+过户费万0.1 |
| 最小单位 | 100股整数倍 |
| 风控 | 单票20%上限、总仓位80%、止损8%、止盈20%、回撤15%暂停 |

## 数据源

- **主数据源**: AKShare（免费，A股数据全）
- **降级数据源**: BaoStock（AKShare不可用时自动切换）
- **Tushare Pro**: 需配置token，数据质量高
- **YahooFinance**: 港股/美股备用

## 双轨交易架构

系统采用**双轨引擎**设计：

```
┌─────────────────────────────────────────────────────┐
│                   双轨交易引擎                        │
├──────────────────────┬──────────────────────────────┤
│    长线股轨道          │      波段股轨道                │
│  (Long-term Track)   │    (Swing Track)             │
├──────────────────────┼──────────────────────────────┤
│ 自动策略优化           │ 自动市场扫描                  │
│ 策略评分选优           │ 多策略筛选                    │
│ 长期持有 + 做T降成本   │ 短期持有 + 止盈止损            │
│ 自主学习优化参数       │ 固定策略执行                  │
└──────────────────────┴──────────────────────────────┘
         │                          │
         └──────────┬───────────────┘
                    ▼
          ┌─────────────────┐
          │   交易限制器      │
          │  频率/金额控制    │
          └────────┬────────┘
                   ▼
          ┌─────────────────┐
          │    风控模块       │
          │ 止损/止盈/仓位   │
          └────────┬────────┘
                   ▼
          ┌─────────────────┐
          │  模拟盘 / 实盘    │
          └─────────────────┘
```
