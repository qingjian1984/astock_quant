# A股量化交易系统

> 完整的A股量化交易系统，支持双轨交易模式（长线股自动优化策略 + 波段股自动扫描）、模拟盘/实盘切换、多数据源、Web管理面板

## 快速开始

### 1. 安装依赖

```bash
# 核心依赖
pip install -r requirements.txt

# Web界面依赖（可选）
pip install -r requirements_web.txt
```

### 2. 运行回测

```bash
# 单标的回测
python backtest/run.py --symbol 000001 --start 20240101 --end 20241231

# 多标的回测
python backtest/run.py --symbol "000001,600519,000858" --start 20240101 --end 20241231

# 切换策略
python backtest/run.py --symbol 000001 --strategy rsi
python backtest/run.py --symbol 000001 --strategy macd
python backtest/run.py --symbol "000001,600519" --strategy multi_factor

# 调整资金和滑点
python backtest/run.py --symbol 000001 --capital 500000 --slippage 0.002
```

### 3. 回测结果

回测完成后在 `results/` 目录生成：
- `equity_curve.png` - 净值曲线 + 最大回撤图
- `trades.png` - 交易信号标记图
- `monthly_returns.png` - 月度收益热力图
- `equity.csv` - 净值数据
- `trades.csv` - 交易明细

## Web管理面板 操作手册

### 启动方式

```bash
cd /root/astock_quant
python web/app.py
```

启动后浏览器访问：`http://localhost:5000`

如需外部访问：修改 `app.py` 最后一行的 `host` 参数，或直接用 `0.0.0.0`（已默认配置）。

### 面板功能总览

Web面板共 **8个标签页**，所有核心功能均可在页面上操作，无需命令行：

#### 1. 总览页

6个关键指标卡片 + 交易模式快速切换：
- 交易模式（模拟盘/实盘）| 长线股数量 | 波段最大持仓
- 扫描间隔 | 日最大交易次数 | 止损比例

#### 2. 回测页（核心功能）

**完整回测流程，无需命令行：**

1. **参数配置：** 股票代码（支持多只逗号分隔）、日期范围、策略选择（双均线/RSI/MACD/多因子）、初始资金、滑点、数据源
2. **运行回测：** 点击按钮后实时显示进度条和运行日志
3. **查看结果：** 回测完成后自动展示：
   - 12个统计卡片（总收益率、年化收益、最大回撤、夏普比率、胜率、交易次数等）
   - 交易明细表格（日期/代码/方向/价格/数量/手续费/原因）
   - 净值数据表格

**异步任务：** 回测在后台线程运行，不阻塞页面，每秒刷新进度。

#### 3. 结果页（4个子标签）

- **图表：** 查看回测生成的图片（净值曲线、交易信号图、月度收益热力图）
- **交易记录：** 加载并查看 trades.csv 完整交易明细
- **净值数据：** 加载并查看 equity.csv 净值曲线数据
- **文件列表：** 列出 results/ 目录下所有文件，支持在线查看

#### 4. 长线股管理

添加股票后系统自动回测寻找最优策略，长期跟踪做T降成本。

| 操作 | 说明 |
|------|------|
| 添加 | 输入代码/名称/资金，点击添加 |
| 优化 | 点击"运行策略优化"，自动回测多策略选最优 |
| 删除 | 从池中移除 |

优化结果：最优策略名称、评分、优化时间，均显示在表格中。

#### 5. 波段股配置

自动扫描选股参数配置：

| 参数 | 说明 | 建议值 |
|------|------|--------|
| 扫描间隔 | 扫描市场频率（秒） | 1800~7200 |
| 最大持仓数 | 同时持有的波段股数量 | 3~10 |
| 止损/止盈 | 自动卖出触发比例 | -5% / +10% |
| 选股策略 | 扫描使用的策略 | ma_cross,macd |

#### 6. 配置页（4个子标签）

**交易限制：** 频率限制（日最大次数、最小间隔）+ 金额限制（单笔/日总额/买卖上限）

**交易参数：** 佣金费率/最低佣金/印花税率/过户费率/滑点/最小交易单位

**风控配置：** 单票仓位/总仓位上限/止损/止盈/最大回撤暂停/黑名单

**数据源管理：** 启用/禁用开关 + Tushare Token + 连接测试

所有配置修改后点击保存，持久化到 `config/web_config.json`。

#### 7. 数据页（4个子标签）

**数据下载：** 从数据源下载指定股票的历史数据，显示行数、日期范围、数据预览

**数据源对比：** 同一股票同一时段，对比多个数据源的数据条数、日期范围、缺失值

**股票列表：** 加载A股全市场股票列表，支持实时搜索（代码/名称），可一键填入回测

**批量回测：** 对同一股票同时运行多个策略（双均线+RSI+MACD），对比结果表格

#### 8. 日志页

查看系统运行日志，可选行数（100/200/500/1000），支持自动刷新（每5秒）

## 策略说明

| 策略 | 文件 | 信号逻辑 | 适用场景 |
|------|------|----------|----------|
| 双均线 | ma_cross.py | MA5上穿MA20买入，下穿卖出 | 趋势行情 |
| RSI | rsi.py | RSI<30超卖买入，>70超买卖出 | 震荡行情 |
| MACD | macd.py | DIF上穿DEA买入，下穿卖出 | 趋势行情 |
| 多因子 | multi_factor.py | 动量+波动率+成交量综合评分 | 组合选股 |

## A股特殊处理（已实现）

| 规则 | 处理方式 |
|------|----------|
| T+1 | 买入当天禁止卖出，记录buy_date |
| 涨跌停 | 涨停(>=+9.9%)不能买，跌停(<=-9.9%)不能卖 |
| 滑点 | 买入价上偏、卖出价下偏，默认千1 |
| 手续费 | 佣金万2.5(最低5元)+印花税千1(卖出)+过户费万0.1 |
| 最小单位 | 100股整数倍 |
| 风控 | 单票20%上限、总仓位80%、止损8%、止盈20%、回撤15%暂停 |

## 数据源

- **主数据源**: AKShare（免费，A股数据全）
- **降级数据源**: BaoStock（AKShare不可用时自动切换）
- **Tushare Pro**: 需配置token，数据质量高
- **YahooFinance**: 港股/美股备用

## 双轨交易架构

系统采用**双轨引擎**设计：

```
┌─────────────────────────────────────────────────────┐
│                   双轨交易引擎                        │
├──────────────────────┬──────────────────────────────┤
│    长线股轨道          │      波段股轨道                │
│  (Long-term Track)   │    (Swing Track)             │
├──────────────────────┼──────────────────────────────┤
│ 自动策略优化           │ 自动市场扫描                  │
│ 策略评分选优           │ 多策略筛选                    │
│ 长期持有 + 做T降成本   │ 短期持有 + 止盈止损            │
│ 自主学习优化参数       │ 固定策略执行                  │
└──────────────────────┴──────────────────────────────┘
         │                          │
         └──────────┬───────────────┘
                    ▼
          ┌─────────────────┐
          │   交易限制器      │
          │  频率/金额控制    │
          └────────┬────────┘
                   ▼
          ┌─────────────────┐
          │    风控模块       │
          │ 止损/止盈/仓位   │
          └────────┬────────┘
                   ▼
          ┌─────────────────┐
          │  模拟盘 / 实盘    │
          └─────────────────┘
```

## API 接口文档

| 方法 | 路径 | 说明 |
|------|------|------|

|------|------|------|
